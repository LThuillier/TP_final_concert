# Maintenance Evolutive Et Corrective

## Corrective
- Reproduire l'erreur avec un test.
- Corriger dans une branche `bugfix/*`.
- Verifier via `php artisan test`.

## Evolutive
- Ajouter les besoins dans le backlog Agile.
- Estimer l'effort (story points).
- Integrer en sprint avec revue de code.

## Verification Apres Livraison
- Routes web et API disponibles.
- Migrations et seeders executables.
- Build frontend valide (`npm run build`).

## Checklist Rapide
- Logs applicatifs surveilles (`storage/logs`).
- Dependances mises a jour.
- Documentation `docs/` tenue a jour.
